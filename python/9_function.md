## Function

- 반복되는 코드에 이름을 붙여 재사용할 수 있도록 하는 것

- 함수 원형
    ```python
    def function(parameter1, ...):
        command1
        command2

    return result
    ```
- 여러개의 결과를 돌려받는 법

    ```python
    def function(parameter1, parameter2):
        command1
        command2

    return result1, result2
    ```

- 함수의 반환이 2개여도 2개로 날아오는게 아니라 하나의 튜플로 던진다
    - ex) (result1, result2)의 형태로
