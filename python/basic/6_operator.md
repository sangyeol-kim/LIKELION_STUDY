### 연산자 (Operator)

## 1. Assign (할당 연산자)
    - =, +=, -=, *=, /=

## 2. Arithemetic (산술 연산자)
    - +, -, *, /
    - **, //(몫), %(나머지)

## 3. String (문자열 연산자)
    - - +, *

## 4. Comparision (비교 연산자)
    - ==, !=, >, <, >=, <=

## 5. Logical (논리 연산자)
    - and, or, not, 

## 6. Membership (멤버쉽 연산자)
- in, not in
    ```python
    >>> str = [1, 2, 3]
    >>> 3 in str
    >>> True
    ```
